üìç TO HIDE FILES IN THE FILE EXPLORER 

  üñä Create a folder named ".vscode" in the root folder.
  üñä Inside that folder, create a file named "settings.json".


  {
     "files.exclude": {
    "**/node_modules": true,   //directory
    "**/index.html": true,    //specific file
    "**/*.pem": true,         //All files with the specific ext

    }
  }

üìç TO INCLUDE HIDDEN FILES IN THE QUICK OPEN SEARCH

  {
     "files.exclude": {
      "**/node_modules": true,   //directory
    },
     "search.exclude": {
      "**/App.jsx": false,   //add files here and set it to false
  }
  }
