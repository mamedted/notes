**TO PUT YOUR  LOCAL PROJECT IN GITHUB**
1. In your project root folder, run git init  //it will create a .git folder, check it if its created.
2. Since you are working with frontend and backend, create a single .gitignore in the root folder.
3. After staging and commit and pushing it, you can pull it from other machine. Just run npm install in frontend and backend separately.
