USING EXPO

1. RUN in cmd: npx create-expo-app@latest myFirstApp

2. cd myFirstApp

3.To start compiling and viewing from mobile, run: npx expo start

4. It will generate a QRC Code. To open, scan it by downloading Expo Go app from playstore.

5. TO RESET THE TEMPLATE, run: npm run reset-project

6. TO HIDE THE UPPER HEADER/TAB: go to _layout.tsx and incorporate this:
export default function RootLayout() {
  return <Stack screenOptions={{ headerShown: false }}></Stack>;
}
