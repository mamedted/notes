1. In the root folder, run
	npm install -d tailwindcss@3 postcss autoprefixer

2.npx tailwindcss init -p
   this creates tailwind.config.js and postcss.config.js

3.Then update config tailwind.config.js by adding in the contemt []

content: [
"./index.html",
"./src/**/*.{js,ts,jsx,tsx}",
],

4.Then in your index.css from src import the ff;
@tailwind base;
@tailwind components;
@tailwind utilities;

5.Make sure index.css is imported in app.jsx

That's it!!!
