• The asterisk "*" usually represents all or zero or more of something.


• Method to link a.js to b.js 
   1.require() CommonJS, traditional
   a.js
    module.exports = {
        key: "1234",
        pass: "34443"
    }

    b.js 
     const config = require("./a") //not require ext
     console.log(config.key)


    2. Import ES modules
     a.js  export const key = 1234
     b.js  import { key } from "./a.js"  //require .js ext

     Note: Say you have a lot of var or function you want to import from the same script
     You can import all by using 
     import * as name from "./a.js" 
     to access it individually, just ff.name   //name of function or variable
